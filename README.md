# Transports Website

This is a transportation website project built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Navigation menu
- Services section
- Contact information

## Getting Started

1. Clone this repository:
```bash
git clone https://github.com/anshpanchal575-lgtm/Transports-Website.git
```

2. Navigate to the project directory:
```bash
cd Transports-Website
```

3. Open `index.html` in your web browser

## File Structure

- `index.html` - Main HTML file
- `styles.css` - CSS styling
- `README.md` - Project documentation
